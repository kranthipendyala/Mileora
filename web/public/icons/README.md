# PWA icons

Replace these placeholders before launch:

- `icon-192.png` (192×192, maskable)
- `icon-512.png` (512×512, maskable)
- `apple-touch-icon.png` (180×180)
- `favicon.ico` (multi-size: 16, 32, 48)

Generate from a single 1024×1024 source via [https://realfavicongenerator.net](https://realfavicongenerator.net) or `pwa-asset-generator`.
